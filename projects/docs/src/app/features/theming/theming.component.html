Theming

The theming system allows you to customize the look and feel of your application by modifying CSS variables. These variables control colors, borders, and other visual aspects of the UI.

How it works

The library uses CSS variables defined in the :root and body elements to manage themes. By default, two themes are provided: light and dark. You can switch between these themes by setting the data-theme attribute on the body element. Each component in the library comes with a very simple default theme, which is based on the light theme. This ensures that components look consistent out of the box, but you can easily customize or override these styles to match your application's design. You can also override individual CSS variables to create a custom theme. The Theming component provides an interface to dynamically update these variables. The theme.css file now includes support for light and dark color schemes using the color-scheme property. It defines base colors, properties, and a default theme that dynamically adjusts based on the data-theme attribute. The light-dark function is used to define values for both light and dark themes, ensuring seamless transitions between themes.

Default Variables

Variable Description
--bg-color Background color of the application.
--text-color Text color used throughout the application.
--background Background color for UI elements.
--foreground Foreground color for UI elements.
--border-color Border color for UI elements.
--primary Primary color used for buttons and highlights.
--primary-foreground Text color for primary elements.
--primary-border-color Border color for primary elements.
--secondary Secondary color used for less prominent elements.
--secondary-foreground Text color for secondary elements.
--secondary-border-color Border color for secondary elements.
--input-background Background color for input fields.
--input-foreground Text color for input fields.
--error-foreground Text color for error messages.
--error Background color for error messages.
--radius Border radius used for rounded corners in UI elements.
